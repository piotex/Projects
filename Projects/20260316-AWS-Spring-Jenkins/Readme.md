### Code
1.1 Copy basic app code
Open: https://start.spring.io/
    Project: Maven
    Language: Java
    Spring Boot: 4.0.3
    Group: com.example
    Artifact: demo
    Name: demo
    Package: com.example.demo
    Packaging: Jar
    Configuration: Properties
    Java: 21
    (Generate)

unzip demo.zip
cp demo repo/demo

1.2. Add Controller and pom.xml dependency
```
vi src/main/java/com/example/demo/HelloController.java
vi pom.xml
```

1.3 Compile and run
```
sudo apt update -y
sudo apt install -y maven openjdk-21-jdk 
```
```
cd demo/
mvn clean package
java -jar target/demo-0.0.1-SNAPSHOT.jar
```

### AWS CLI

    User name: tech-user
    Provide user access to the AWS Management Console (check)

    Attach policies directly

    Create User
