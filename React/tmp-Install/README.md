# React 

#### Install
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
source ~/.bashrc
nvm install --lts
node -v
npm -v
```

#### Run locally
```
npx create-react-app react-ts-basics --template typescript
cd react-ts-basics
npm install
npm start
```

#### Compile
```
npm run build
```
